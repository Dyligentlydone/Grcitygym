# Grand Rapids City Gym

Modern marketing site + member portal, Equinox-inspired black minimal aesthetic with a red accent.

## Stack
- Next.js 14 (App Router) + TypeScript
- Tailwind CSS
- Framer Motion
- Lucide icons

## Run
```bash
npm install
npm run dev
```

Open http://localhost:3000

## Structure
- `src/app` — routes (marketing + `/portal` member area)
- `src/components` — UI primitives
- `src/lib/data.ts` — placeholder content (replace with real copy/assets)

## Deploy
Compatible with Vercel or Railway out of the box.
